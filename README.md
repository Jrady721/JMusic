# JMusic

#### 간단한 설명
```
음원 앨범을 장바구니에 담구 구매할 수 있는 사이트입니다.
React 를 통해 프론트엔드 작업을 하였습니다.
장바구니 기능이 구현되었습니다.
검색기능과 하이라이트 기능을 구현하였습니다.
실시간으로 데이터 처리를 하였습니다.
(예 - 앨범수량을 늘리면 자동으로 장바구니의 수량이 즉시 올라가는 등...)
```

사용한 툴
```
// JSX 빌드
npx babel --watch src --out-dir ./assets/js --presets react-app/prod

// 더미 CSS 삭제
purifycss assets/css/custom.css assets/js/script.js  --min --info --out test.css
```
